# raid_bot
show an overview of all raid on telegram

#### HELP ON DISCORD: https://discord.gg/sDxjY7M

### Install Telebot

`pip3 -r install requirements.txt`

### Config
`cp config_example.ini config.ini`

### Start
`python3 raid-main.py`
